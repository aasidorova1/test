# my-academic-cv-site

This site was created with **Ownable CMS** using the **Academic CV** template.

## Getting Started

1. **Install Dependencies**
   ```bash
   pnpm install
   ```

2. **Start Development Server**
   ```bash
   pnpm run dev
   ```

3. **Build for Production**
   ```bash
   pnpm run build
   ```

## Template Features

- Publication lists
- Research sections
- Academic timeline
- Contact forms

## Perfect For

- Professors
- Researchers
- PhD students
- Academic professionals
